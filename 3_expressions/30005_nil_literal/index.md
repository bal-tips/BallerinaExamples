The `nil` basic type has only a single value called `nil`. Nil literal uses the syntax `()` and `null`.


**Syntax** : `()` | `null`

ðŸ’¡ Note: Both `()` and `null` represent the nil value in Ballerina. However, `()` is the preferred way to assign **nil** value to a variable.
